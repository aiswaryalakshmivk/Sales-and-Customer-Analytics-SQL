# Sales and Customer Analytics SQL

## Overview
This project focuses on business analytics using SQL for customer, sales, and revenue analysis.

It includes SQL queries to analyze:
- Customer information
- Sales performance
- Top-selling products
- Monthly revenue trends

## Files Included

- `customer_analysis.sql` – Customer-based SQL analysis
- `sales_analysis.sql` – Sales-related SQL queries
- `top_products.sql` – Product performance analysis
- `monthly_revenue.sql` – Revenue trend analysis
- `customers.csv` – Sample customer dataset
- `sales.csv` – Sample sales dataset
- `products.csv` – Sample product dataset

## Skills Used
- SQL
- Joins
- Aggregate Functions
- GROUP BY
- ORDER BY
- Subqueries
- Business Analytics Concepts

## Purpose
To practice SQL queries and apply them to real-world business analytics scenarios.
